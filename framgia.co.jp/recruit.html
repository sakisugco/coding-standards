募集職種

ソリューションセールス(営業)

業務内容

企業の経営層、人事担当者、実務担当者様などできるだけ経営や実務に近い方々とお会いし、会社のビジョン、経営戦略、現状課題をしっかりと理解した上で、最適なソリューションをご提案します。

課題分野に精通したコンサルタントをアサインしてアドバイザリーサービスをご提供することもあれば、外部パートナーとの混成チームを組成して、戦略の実行を請け負うこともあります。

また、弊社でアドバイザリーやエグゼキューションといった業務を請け負うだけではなく、課題解決のために中長期的にクライアント内部に必要と思われるポジションについては、弊社のHRソリューション部隊を使って最適な人材のサーチし、クライアントへご紹介することもあります。

クライアントの課題とそれを解決できるソリューションサービスや人材を結び付ける非常に重要な仕事になります。
クライアントのニーズから発想して、新たな商品・サービスを生み出していくことにも積極的に関与していただきます。

必須スキル
  • 法人営業の経験がある方（3年以上）
歓迎スキル
  • プレゼンテーションスキルが得意な方（企業へサービスの提案など、経験が豊富な方）
  • 様々な企業の経営層、マネージメント層と一緒に仕事をした経験がある方
求める人物像
  • 社会にインパクトを与える事がしたい方、人に感謝されるような仕事がしたい方、
    企業家精神がある方
  • 自主的に学ぶことに意欲的な方
  • 積極的に発言、提案ができる方

キャリアコンサルタント

業務内容

当社のキャリアコンサルタントとは？お一人お一人と向き合って、今後のキャリアを考える上での何らかの「気付き」や「きっかけ」をご提供し、重要な意思決定のサポートをします。

具体的には、転職希望者の方々と面談をし、まずは、これまでのご経験やご希望のヒアリングをします。そして、目指すキャリアゴールに向かって今どのようなキャリアアップ、ステップが必要か、そのためには何をいつまでにする必要があるのか、的確なアドバイスをします。

決して無理に転職を勧めるのでなく、そもそも転職すべきかどうかまでを含めた、仕事を中心とした人生設計の相談にのり、長期的な関係を築いていきます。

また、本人のやりたいことや目指すものに対して足りない部分があれば、それをきちんと指摘し、現実の厳しさをきちんと伝えていくことも大切になります。

しかしながら、難しくてもすぐに転職を諦めさせる方向へ導くのではなく、難しいということを認識してもらった上で、目指すキャリアゴールへの道筋を一緒に考え、ゴールに近づくためにできる限りのバックアップを行います。

創業間もない会社だからこそ業界経験豊富な先輩社員から直に学ぶことができるので成長意欲の高い方には魅力的な環境になるはずです！先輩のフォローを受けながらキャリアコンサルタントとして早い段階で一人前を目指していただきます。

必須スキル
  • 人材紹介、派遣でのキャリアコンサルタント経験がある方
歓迎スキル
  • 専門分野における実務経験（経理・財務、セールス、マーケティング、法務、人事、
    IT、エンジニアリング、サプライチェーン） がある方
求める人物像
  • 社会にインパクトを与える事がしたい方、人に感謝されるような仕事がしたい方
  • チームプレイヤー
  • コミュニケーション能力（聞く力）が高い方
  • ソーシャルメディアなどを利用して、人脈を広げる、ネットワーキングができる方
  • 自主的に学ぶことに意欲的な方・積極的に発言、提案ができる方

エンジニア

業務内容

【主な業務】
人材マッチングサービスを主軸に置きながら、技術的・科学的に最先端な要素を取り入れ、革新的なサービスを創造しています。

設計から開発まで、自社サービスの開発を仲間とともに取り組んで頂きます。

具体的には、
・システム設計
・サービス開発 - サーバサイド - フロントエンド
・インフラ整備

お願いする仕事内容はあなたの強みや興味に応じて相談の上決めることになります。

【技術】 メインの言語はPHPを使用しています。
サブとしては、Pythonやjavascript(node.js)
を使っています。HTML5の技術を中心に新しいことにも挑戦していただけます。

必須スキル
  • Web開発経験のある方（２年以上）
  • LAMP環境での開発スキル
  • Javascript
  • GIT
歓迎スキル
  • Nginx
  • Redis
  • Phalcon(PHP)
  • スマートフォンアプリの開発
  • Ruby/Python等のスクリプト言語
  • Chef
求める人物像
  • 基本的なWebサービスを生み出したい方
  • 開発だけでなく企画〜運用まで幅広いエンジニアリングを行いたい方
  • 既存の技術にとらわれることなく新しい技術に挑戦したい方
  • 会社に縛られないエンジニアらしい働き方をしたい方

Webディレクター

業務内容

私達は「WEBサイトを制作する」という短期的な付き合いではなく、クライアントのビジネス課題や事業構想を理解し、長期的にWEBサイトを通したビジネスを支援し事業の成功に導くことを目指しています。

当社のクライアントへのコミットは一人ですべてを完結できるものではありません。あなたの持ち味と私達が培ったノウハウをうまく融合することで活躍していただけるステージを作ることができるはずです。

「様々な業種のWEBサイトの運営に携わってスキルアップしたい」、 「クライアントの信頼を得て長くお付き合いできるサポートをしたい」、「能力の高い仲間とチームで仕事がしたい」という方であればきっとやりがいを感じていただけるはずです。

必須スキル
  • WEBディレクター経験3年以上
歓迎スキル
  • 告知文などの、日本語文章の制作が得意な方
  • BtoCサービスのWEBサイト運営に関わったことがある
  • HTML、CSS、JavaScript等の基本的知識をお持ちの方
  • WEBコンサルや事業会社でサイトの企画・ディレクションから、ログ解析、改善提案などサイト運営のPDCAを一通り回してきた経験がある方
求める人物像
  • 自由度の高い仕事スタイルを追求する人や専門分野（得意な業務領域）を伸ばしたい方
  • チームプレイヤー
  • コミュニケーション能力（聞く力）が高い方
  • ソーシャルメディアなどを利用して、人脈を広げる、ネットワーキングができる方
  • 自主的に学ぶことに意欲的な方・積極的に発言、提案ができる方

SEOコンサルタント

業務内容

あるべきSEO戦略の立案から、WEBサイトのボトルネックを抽出し、具体的な改善案を提案するといったコンサルティング業務全般を行って頂きます。

日本では正しい知識を持ったSEOコンサルタントが少なく、誤った施策を行っているWEBサイトが散見している状況です。

SEOの本質を理解し、本当に必要な「成果」を出すための仕掛けをWEBサイトに施すことが主な業務となります。時には提案内容がWEBマーケティング全体に及ぶこともあります。

自らの知識と経験を活かして、トコトン成果にこだわって取り組んでい頂ける環境を約束します。

必須スキル
  • SEOコンサル、WEBサイト分析から改善指示提案までの経験がある方（1年以上）
歓迎スキル
  • WEBサイトの制作、集客、解析など、WEBマーケティング全般の経験がある方
  • Excel、PowerPoint等の基本的なPCスキルがある方
  • HTML、CSSに関する基本的な知識のある方
求める人物像
  • 論理的思考能力、問題解決力のある方
  • コミュニケーション能力の高い方
  • 自主的に学ぶことに意欲的な方・積極的に発言、提案ができる方
